Chowdeck

Project Description

What is this project about?



This group project is about designing an interactive interface for an emerging startup, which aims to solve the problem the startup is working on. Chowdeck is what our group choose for this project. It is an on-demand food delivery mobile App like Uber Eats, but only for Africa. Its mission is to empower restaurants in Africa to deliver meals to customers anywhere and anytime, and customers can buy food in a few short steps and have it delivered to their doorstep in under 30 minutes.

We used Figma to complete this project.




What problem does this project solve?



  • We made every page in our app with a very clean and clear overall layout, so that customers are more likely to find what they are looking for.
  • We included a function in our app which customers can checkout all the orders from different restaurants in cart, so that it saves a lot of time checking out one by one.
  • We minimize the number of steps the customers needs to click thoughout the whole ordering process.

Some Things I Learned



  1. It's very helpful to talk to other people about your design, and gather their opinions and suggestions.
  2. Don't design somthing only because it's beautiful, but because it is useful and problem solver.
  3. Don't be afraid to ask questions if you encounter some problems.
