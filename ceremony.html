Ceremony

Project Description

What is this project about?

This individual project is about redesigning a existing website. The main task is to find the problems of the existing website, and fix them or make them better. Ceremony Providence's website is what I choose for this project. It is an independently owned caf√© and tea house in providence focused on bringing customers a unique experience.

I used HTML and CSS to complete this project.



What problem does this project solve?



  • I made all the categories into buttons instead of very thin and small texts like the original page, so that people can know it is actually a button and it is clickable.
  • I made the website highlight which category you are currently on instead of not showing it like the original page. So that it is easy for users to navigate though the page.
  • I also highlight the order online button because most people go to their website to order drinks online.
  • In the phone screen size, I made the menu text into a button under the ceremony title instead of a tiny text at the top center of the page. Therefore, people can easily find it and navigate though the page.

Some Things I Learned



  1. When design a user interface, it is crucial to stand in customers' shoes to find out what they will normally do.
  2. Do not run your HTML after you wrote a lot of stuff, take a short step at a time, run it as you go, so that it will be easy to debug when something is wrong.
  3. Start ahead of time, something small or seems easy may take a long time to code.
