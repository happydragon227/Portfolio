Bakery Shop

Project Introduction

What is this project about?



This individual project is about Developing an interactive app using React. Therefore, the main focus of this project is about the functionalities, not the overall design. I choose to create a bakery app that enables customers to order food more efficiently. The main functionalities of this app include: filtering by categories, sorting by feature, an aggregator showing the total price and what item you have added to the shopping cart, and 15 bakery items in total. Users can apply none or multipule filters or sorting features at the same time to find the items they want to buy.

I used HTML, CSS, JavaScript, and React to complete this project.

How to use React to build this App:

  • I used the React "State Hook" to achieve all the functionalities this app has. The app just displays what's inside the State. In this way, we do not need to modify the HTML component and get the interactive results by modifying what's inside the State.

What problems can this App solve:

  • Customers can use the "types of bakery" filter and "Dietary Restrictions" filter to find the bakery items they want to buy faster.
  • Customers can use the sort function to sort all the bakery items either by prices or ratings, so that it is more convenient for the customers to find what they want.
  • Customers can also add or remove items from the shopping cart if they want.


Some Things I Learned



  1. Using React API from Material UI really helps a lot. You can use the existing component from this website and it really saves a lot of time.
  2. State updates in React are asynchronous. When an update is requested, there is no guarantee that the updates will be made immediately. The updater functions enqueue changes to the component state, but React may delay the changes, updating several components in a single pass.
  3. Writing each component in different files instead of writing everything in the app.js helps a lot. It is a lot easier to debug and reuse the function component.
