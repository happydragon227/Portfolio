Coca Cola
Fountain Machine

Project Description

What is this project about?

This individual project is a classic UX task, which is about finding a real world interface, observing real users interacting with an interface, interviewing these individuals about their experiences, Construct 2 personas by creating a four-quadrant empathy map, describing what users think, feel, say, and do, and illustrating a storyboard for one of the personas. Coca Cola Soda Fountain Machine is the real world user interface I choose for my project.

I used Boords, Figma, and Powerpoint to complete this project.




What problem does this project solve?



  • I also create 2 personas and storyboard to help users to understand more vividly about the ups and downs of this fountain machine.
  • Maybe the customers sees it, they will know better how to avoid some inconvenience this machine brings.
  • Maybe the manufacture sees it, they will be able to redesign or improve this fountain machine so that these problems will not occur again.



Some Things I Learned



  1. Asking the same question differently will actually result in different answer. In some circumstances, people may lie because of the way you asked this question, or the environment they are in.
  2. Do not just think about what other people might do, actually go outside and observe their actions.
  3. Don't just observe their action on the same time together, because they might be the same group of people, and the overall pattern you get will not be very comprehensive. Try to observe them in different time around the day.
